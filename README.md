# 😷 Face Mask Detection System

## 📌 Overview

The Face Mask Detection System is a Computer Vision and Deep Learning project developed to detect whether a person is wearing a face mask or not in real-time using a webcam or image input.

This project uses OpenCV and TensorFlow to perform face detection and mask classification, helping in understanding real-time image processing and deep learning applications.

---

## 🚀 Features

* Real-time face mask detection
* Detects Mask and No Mask faces
* Webcam support
* Image preprocessing
* Deep Learning model training
* Real-time prediction display
* User-friendly detection system

---

## 🛠️ Technologies Used

* Python
* OpenCV
* TensorFlow
* Keras
* NumPy
* Matplotlib
* Scikit-learn

## 📂 Dataset

The dataset used contains images of people:

* Wearing masks
* Not wearing masks

The dataset was used for training and testing the deep learning model.
